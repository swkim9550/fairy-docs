개인정보 처리방침

최종 수정일: 2025년 11월 28일

1. 개요

Fairy(이하 "앱")는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 개인정보 처리방침은 앱이 어떻게 정보를 수집, 사용, 보호하는지 설명합니다.

2. 수집하는 정보

2.1 로컬 저장 데이터

앱은 다음 정보를 사용자 기기에만 로컬로 저장합니다:

  • 일기 내용
  • 감정 기록
  • 요정 캐릭터 선택
  • 퀘스트 진행 상황
  • 앱 설정

이 데이터는 외부 서버로 전송되지 않습니다.

2.2 Google Analytics

앱은 Google Analytics를 통해 다음의 익명화된 데이터를 수집합니다:

  • 앱 사용 통계 (세션 수, 활성 사용자)
  • 기능 사용 현황
  • 앱 충돌 정보
  • 기기 정보 (OS 버전, 기기 모델)

개인을 식별할 수 있는 정보는 수집하지 않습니다.

3. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 앱 성능 개선
  • 사용자 경험 향상
  • 버그 및 오류 해결
  • 기능 개발 우선순위 결정

4. 정보 보호

4.1 로컬 데이터

  • 모든 일기 및 개인 데이터는 사용자 기기에만 저장됩니다
  • 사용자가 앱을 삭제하면 모든 데이터가 삭제됩니다
  • 백업 기능을 사용하는 경우, 사용자가 암호화 여부를 선택할 수 있습니다

4.2 Google Analytics 데이터

  • Google의 보안 정책에 따라 보호됩니다
  • 익명화되어 개인을 식별할 수 없습니다

5. 제3자 공유

앱은 다음을 제외하고 개인정보를 제3자와 공유하지 않습니다:

  • Google Analytics: 익명화된 사용 통계만 공유
  • 법적 요구사항이 있는 경우

6. 사용자 권리

사용자는 다음 권리를 가집니다:

  • 앱 삭제를 통한 모든 로컬 데이터 삭제
  • Google Analytics 추적 거부 (기기 설정에서 광고 ID 초기화)
  • 개인정보 처리에 대한 문의

7. 문의

개인정보 처리에 대한 문의는 다음으로 연락주세요:

  • 이메일: kbkkim2@naver.com

8. 정책 변경

본 개인정보 처리방침은 예고 없이 변경될 수 있습니다. 변경 시 앱 내에서 공지합니다.

Privacy Policy

Last Updated: November 28, 2025

1. Overview

Fairy (hereinafter "App") prioritizes the protection of user privacy. This Privacy Policy explains how the App collects, uses, and protects information.

2. Information We Collect

2.1 Local Storage Data

The App stores the following information locally on the user's device only:

  • Diary entries
  • Emotion records
  • Fairy character selection
  • Quest progress
  • App settings

This data is never transmitted to external servers.

2.2 Google Analytics

The App collects the following anonymized data through Google Analytics:

  • App usage statistics (sessions, active users)
  • Feature usage patterns
  • App crash information
  • Device information (OS version, device model)

No personally identifiable information is collected.

3. How We Use Information

Collected information is used solely for:

  • Improving app performance
  • Enhancing user experience
  • Fixing bugs and errors
  • Prioritizing feature development

4. Data Protection

4.1 Local Data

  • All diary and personal data are stored only on the user's device
  • All data is deleted when the user uninstalls the app
  • If backup features are used, users can choose encryption options

4.2 Google Analytics Data

  • Protected under Google's security policies
  • Anonymized to prevent personal identification

5. Third-Party Sharing

The App does not share personal information with third parties except:

  • Google Analytics: Only anonymized usage statistics
  • When legally required

6. User Rights

Users have the right to:

  • Delete all local data by uninstalling the app
  • Opt-out of Google Analytics tracking (reset advertising ID in device settings)
  • Contact us regarding data processing

7. Contact Us

For inquiries about data processing, please contact:

  • Email: kbkkim2@naver.com

8. Policy Changes

This Privacy Policy may be updated without notice. Changes will be announced within the App.
